# Operational Resilience

## Purpose

Operational resilience ensures automation improves recovery without creating new failure modes.

## Resilience Topics

- safe automation
- fallback paths
- service validation
- escalation handling
- recovery confidence
- bounded failure modes

## Evidence To Collect

- validation results
- fallback procedures
- recovery test notes
- exception logs

## Use

Operational resilience is strongest when automation improves recovery without reducing visibility or control.
